About this ICU EEG Dashboard

The vision for this dashboard is simple: to aid physicians in promptly digesting & triaging EEG results from patients currently bedded in the HUP. A patient's data can be selected to be viewed from the home page, where each patient is delineated by bed number: RID00BED_NUMBER

The dashboard streams patient data from iEEG, outputting a prediction for each 10-second time window on the likelihood that a seizure occurred during that period.

The color coding goes as follows:

  • Blue => Likely healthy period

  • Yellow => Potential seizure

  • Red => Likely seizure
